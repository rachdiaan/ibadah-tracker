1. Pendahuluan

1.1 Tujuan Aplikasi

Sebagai sebuah karya digital yang inovatif dari Rachdian Habi Yahya, aplikasi ini hadir dalam bentuk Progressive Web App (PWA) canggih yang dirancang untuk memperkaya pengalaman spiritual harian Anda. Aplikasi ini secara dinamis mengintegrasikan amalan dzikir Al-Ma'tsurat Sugro dengan informasi jadwal shalat yang akurat untuk wilayah Bandung, disajikan melalui antarmuka liquid glass yang modern dan sangat responsif. Tujuannya adalah untuk memberikan pengalaman yang tidak hanya menenangkan, tetapi juga fungsional dan memberdayakan.

Aplikasi ini sepenuhnya mandiri (client-side) dan tidak memerlukan server backend untuk fungsi utamanya. Fitur tambahan seperti refleksi (Tadabbur) ditenagai oleh Google AI (Gemini API).

2. Fitur Utama

  • Laman Utama Dinamis: Menampilkan widget jadwal shalat interaktif dengan jam digital dan tanggal yang berjalan secara real-time.
  • Jadwal Shalat & Countdown: Menampilkan jadwal shalat untuk hari ini (Subuh, Dzuhur, Ashar, Maghrib, Isya) dan hitung mundur (countdown) menuju waktu shalat terdekat berikutnya.
  • Notifikasi Waktu Shalat: Saat waktu shalat tiba, sebuah notifikasi layar penuh yang indah akan muncul dengan animasi untuk mengingatkan pengguna.
  • Dzikir Pagi & Petang: Menyediakan dua halaman terpisah untuk dzikir Al-Ma'tsurat pagi dan petang.
  • Penghitung Tap (Tap Counter): Setiap kartu dzikir memiliki penghitung interaktif untuk membantu pengguna melacak jumlah bacaan. Progress bar visual juga akan terisi sesuai hitungan.
  • Tadabbur dengan AI: Fitur "Tadabbur" pada setiap kartu dzikir yang ditenagai oleh Gemini API untuk memberikan refleksi singkat mengenai makna dan hikmah dari doa atau ayat tersebut.
  • Desain Modern & Responsif: Menggunakan estetika liquid glass ala Apple/iOS, palet warna yang menenangkan, dan animasi yang halus. Tata letak sepenuhnya responsif untuk semua ukuran perangkat.
  • Panel Kontrol Pratinjau: Sebuah panel kontrol di pojok kanan bawah (hanya untuk tujuan pengembangan) untuk memudahkan navigasi ke semua halaman dan fitur utama.

3. Struktur Proyek

Proyek ini dipisahkan menjadi empat file utama untuk keterbacaan, pengelolaan, dan performa yang lebih baik.

3.1 Rincian File

  • index.html: File kerangka utama yang berisi struktur dasar HTML dan menautkan file CSS serta JavaScript.
  • style.css: Berisi semua aturan styling, termasuk variabel warna, efek liquid glass, animasi, dan tata letak responsif.
  • data.js: Menyimpan semua data statis aplikasi, termasuk data jadwal shalat tahunan untuk Bandung dan data lengkap untuk dzikir pagi dan petang.
  • app.js: Merupakan inti dari aplikasi yang berisi semua logika JavaScript, seperti rendering halaman, manajemen event, fungsi countdown, dan panggilan ke Gemini API.

4. Penjelasan Kode

4.1 `index.html`

File ini berfungsi sebagai "tulang punggung" aplikasi.

  • <head>: Bagian ini memuat meta tag standar, tautan ke CDN TailwindCSS, Google Fonts (Amiri & Inter), dan stylesheet lokal style.css.
  • <body>: Berisi elemen-elemen HTML dasar yang akan dimanipulasi oleh app.js, seperti header, main, footer, serta struktur untuk modal dan notifikasi. Tautan ke file JavaScript data.js dan app.js ditempatkan di akhir body.

4.2 `style.css`

File ini bertanggung jawab atas seluruh tampilan dan nuansa aplikasi.

  • :root: Mendefinisikan variabel CSS untuk palet warna agar mudah diubah dan konsisten.
  • .liquid-bg: Menciptakan animasi gradien yang bergerak sebagai latar belakang.
  • .glass-card: Kelas inti untuk efek *liquid glass*, menggunakan background transparan dengan backdrop-filter: blur().
  • .next-prayer-highlight & @keyframes pulse-highlight: Memberikan efek sorotan dan animasi denyut pada kartu waktu shalat berikutnya.
  • .prayer-alert-overlay & @keyframes: Mengatur tampilan notifikasi waktu shalat.

4.3 `data.js`

File ini sangat sederhana, hanya berisi dua variabel JavaScript:

  • const prayerData: Objek besar berisi data jadwal shalat untuk Bandung selama satu tahun.
  • const dzikirDataPagi & dzikirDataPetang: Array objek berisi data lengkap untuk dzikir pagi dan petang.

4.4 `app.js`

Ini adalah file paling kompleks yang menggerakkan seluruh aplikasi.

Logika Utama

  • Selektor DOM: Variabel untuk menyimpan referensi ke elemen HTML utama.
  • Fungsi Rendering Halaman: `renderLandingPage()`, `renderDzikirPage(type)`, `renderAboutPage()` untuk membangun tampilan secara dinamis.
  • Manajemen Event: Mengatur semua interaksi pengguna, seperti klik tombol, tap pada kartu, dll.

Fungsi Kunci

  • `initializePrayerWidget()`: Memulai jam digital dan countdown.
  • `getNextPrayer()`: Logika cerdas untuk menentukan waktu shalat berikutnya.
  • `showPrayerTimeAlert(...)`: Menampilkan notifikasi waktu shalat dengan pesan dinamis.
  • `getTadabbur(...)`: Fungsi async yang memanggil Gemini API untuk mendapatkan refleksi doa.

5. Instalasi & Penggunaan

  1. Simpan Semua File: Pastikan kelima file (`index.html`, `style.css`, `data.js`, `app.js`, dan file dokumentasi ini sendiri dengan nama `documentation.html`) telah disimpan.
  2. Letakkan di Folder yang Sama: Tempatkan semua file tersebut di dalam satu direktori/folder.
  3. Buka `index.html`: Klik dua kali pada file `index.html` untuk membukanya di browser web modern seperti Chrome, Firefox, atau Safari.

Aplikasi akan berjalan sepenuhnya di browser Anda tanpa perlu koneksi internet (setelah semua file dimuat pertama kali), kecuali untuk fitur "Tadabbur" yang memerlukan koneksi untuk menghubungi Gemini API.
